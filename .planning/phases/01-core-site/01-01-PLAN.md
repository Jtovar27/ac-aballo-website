  <action>
Create the directory structure and three core files:

**index.html** — Full HTML5 document shell containing:
1. `<head>` with charset UTF-8, viewport meta, title "AC.aballo — Escuela de Caballos", Google Fonts preconnect + link for Playfair Display (700, italic 400) and Inter (400, 500, 600), Tailwind v4 CDN script (`<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>`), and `<style type="text/tailwindcss">` with the full @theme block (all --color-* and --font-* tokens listed in interfaces above). Also include `html { scroll-behavior: smooth; }` inside the @theme style block.
2. `<body class="bg-neutral-50 font-sans text-earth-600">` with:
   - Fixed nav (see Task 2)
   - Hero section (see Task 2)
   - Five empty section placeholders with IDs: `#philosophy`, `#about`, `#services`, `#contact`, `#footer` — each with a minimum height comment so they're visible during development
3. `<script src="js/main.js" defer></script>` as the last element before `</body>`

**css/styles.css** — Minimal file with:
- A comment header: `/* AC.aballo — Custom overrides. Tailwind v4 CDN handles utilities. */`
- No content yet (placeholder for any animations needed in Phase 2)
- Add `<link rel="stylesheet" href="css/styles.css">` in the index.html head AFTER the Tailwind script

**js/main.js** — Bilingual scaffold:
- Full translations object with `es` and `en` keys covering ALL nav and hero strings (keys: nav.philosophy, nav.about, nav.services, nav.contact, hero.headline, hero.tagline, hero.cta)
- `applyLanguage(lang)` function that queries `[data-i18n]` elements and sets textContent from translations, sets `document.documentElement.lang`, saves to `localStorage`
- Mobile nav toggle: `menuToggle.addEventListener('click', ...)` toggling `hidden` class on `#mobile-menu`
- Auto-close mobile menu when a nav link is clicked
- Init call: `applyLanguage(localStorage.getItem('lang') || 'es')`
- Comment at top: `// Phase 1: translations scaffold. Phase 2 will activate the lang toggle button.`

**images/hero/.gitkeep, images/founders/.gitkeep, images/school/.gitkeep** — Empty placeholder files to create directory structure.

Critical constraints:
- Custom colors MUST be in `@theme { --color-*: value; }` inside `<style type="text/tailwindcss">` — NOT in tailwind.config.js (v4 CDN ignores that file)
- Do NOT use `bg-blue-*`, `bg-purple-*`, or `bg-red-*` anywhere
- `css/styles.css` link goes AFTER the Tailwind CDN script so Tailwind loads first
